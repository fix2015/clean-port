Clean-Port Example

Clean-Port is a Node.js utility for managing processes running on specific ports. Below are examples of how to use it.

Installation

Install the package globally to use it as a CLI tool:

npm install -g clean-port

Or use it directly via npx:

npx clean-port <port>

CLI Usage

Kill a Single Port

npx clean-port 3000

Kill Multiple Ports

npx clean-port 3000 4000 5000

Interactive Mode

List active ports and choose which ones to terminate:

npx clean-port --interactive

Graceful Kill

Attempt a graceful termination before forcefully killing:

npx clean-port 3000 --graceful

Dry Run

Preview the actions without actually killing processes:

npx clean-port 3000 --dry-run

Kill a Range of Ports

npx clean-port --range 3000-3100

Programmatic Usage

You can also use clean-port in your JavaScript/Node.js projects.

Example Code

const cleanPort = require('clean-port');

(async () => {
    try {
        await cleanPort(3000, {
            method: 'tcp',
            graceful: true,
            verbose: true
        });
        console.log('Successfully terminated processes on port 3000');
    } catch (error) {
        console.error('Failed to terminate processes:', error.message);
    }
})();
